# Release v1.0.0 - The Local AI Revolution

We are thrilled to announce the first public release of **DocuMind AI**! This release brings powerful, offline document analysis capabilities directly to your desktop.

- **100% Offline & Private**: All processing happens on your machine. No data is ever sent to the cloud.

- **Integrated OCR**: Built-in Tesseract engine to unlock text from scanned PDFs and images.

### Features

- **Drag-and-Drop Interface**: Easily process PDF, JPG, PNG, and TXT files.

- Packaged as a standalone Windows executable.
